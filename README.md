## About

This project is a simple, fun website created to celebrate Ryan Higa's birthday. It’s a lightweight HTML/CSS/JS project designed for anyone to view and enjoy in their browser — no installation required.

This project was a fun way to display creations from the lamp community to Ryan on his stream.

## Installation

No installation is required. Simply visit the [live site](https://crocross.github.io/RyanHiga2025/) to view it.
<details>
  <summary>Click to reveal wordle solution</summary>

</details>

If you’d like to host it yourself:

1. Clone the repository:
   ```bash
   git clone https://github.com/crocross/RyanHiga2025.git
   ```
2. Navigate into the cloned folder and open `index.html` in any modern web browser.
